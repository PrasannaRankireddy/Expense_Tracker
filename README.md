A simple and powerful Desktop Personal Expense Tracker Application built using Python, Tkinter, SQLite, and Matplotlib. This application helps users record daily expenses, track spending by category, visualize data using charts, and manage a monthly budget.

**Features**

1.Add new expenses (Date, Category, Description, Amount)

2.Update existing expenses

3.Delete expenses

4.Category-wise spending Pie Chart (Matplotlib)

5.Export expenses to CSV (Excel compatible)

6.SQLite database for persistent storage

7.Auto Serial Number (S.No) display

8.Monthly Budget Alert

9.Clean and user-friendly Tkinter GUI

**Technologies Used**

1.Python 3

2.Tkinter (GUI)

3.SQLite3 (Database)

4.Matplotlib (Charts)

5.CSV (Export)

6.Datetime (Auto Date)
